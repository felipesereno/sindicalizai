Este site utiliza um script em node.js para mostrar as notícias do site "A Verdade" em um carrossel na página inicial. Ele é necessário pois os navegadores utilizam uma política chamada de Cross-Origin Resource Sharing (CORS), que impede requisições de sites a outros domínios.
Utilizando um repositório chamado cors-anywhere (https://github.com/Rob--W/cors-anywhere), executo o serviço usando o gerenciador de pacotes 'npm'.
Instruções para iniciar o serviço:
1. Posicione-se no diretório 'cors-anywhere': cd node_modules\cors-anywhere\.
2. Rode o serviço digitando 'npm start'.
